# j1search_es
简单的一个商品搜索项目,根据关键字从从京东爬取部分数据,并推送至es索引中,
此项目主要是为了将es和springboot集成起来,做一个简单的demo.

后面会重构公司搜索项目.将SOA框架重构为SpringBoot,此项目是完整的电商实战搜索项目.
后续新增功能有:
1),排序
  基于商品销量,
  商品评论数,
  商品的上下架时间,
  商品价格
2),分页
  es分页查询,每页只展示16条数据
3),查询类型
 match   查询
 term    查询
 boolean 查询
 range   查询
 phrase  查询
 multi   查询
 filter  查询
4),查询方式
  1,按照关键字查询
  2,按照商品编码查询
  3,按照分类查询
5),聚合
6),搜索提示
